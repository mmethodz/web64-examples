# Web64 Community Projects

This folder contains community-contributed Web64 projects.

Directory layout:

community/
    user-name/
        project-name/
            README.md
            project-name.web64proj
            ...

Each project should include:

- README.md
    - Project description
    - Screenshot (optional)
    - Controls
    - Notes
    - Author

- project-name.web64proj
    - Complete Web64 project

Additional assets may be included if needed.
