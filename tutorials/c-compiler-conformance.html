Web64 C Compiler Conformance

Open ../c-compiler-conformance/c-compiler-conformance.web64proj in Web64 IDE and run _main.

The program prints PASS when the implemented Web64 C v1 expression, cast, call, fastcall-runtime, and printf argument paths behave as expected. A failure prints FAIL and the failing check code.

This tutorial is browser-local and does not require a native compiler, linker, or host filesystem path.
