# Sprite Joy SID Bumps C

A C-first Web64 IDE version of `sprite-joy-sid-bumps`, keeping the animated ghost sprite asset, joystick acceleration, 16-bit X positioning with 320-pixel wrap, the bumps charset backdrop, and the SID tune.

Open `sprite-joy-sid-bumps-c.web64proj` in Web64 IDE, enable audio if the browser asks, and use joystick port 2 or the mapped gamepad/keyboard controls.

The example keeps assembly only where Web64 C still needs it: copying `.incbin` sprite/charset assets from assembler labels into C64 memory.
