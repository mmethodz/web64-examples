# Ghost Game C

A Ghost Game loop based on Web64 IDE version of `sprite-joy-sid-bumps`, keeping the animated ghost sprite asset, joystick acceleration, 16-bit X positioning with 320-pixel wrap, the bumps charset backdrop, and the SID tune.

Uses sub-pixel movement for sprite acceleration and deceleration and has golden spinning coins for the ghost to pick up and gain point.

It is a hybrid project of C and assembly.
