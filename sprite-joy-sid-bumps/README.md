# Sprite Joy SID Bumps

A self-contained Web64 IDE example combining the animated ghost sprite asset, joystick acceleration, 16-bit X positioning with 320-pixel wrap, the bumps charset backdrop, and a small SID tune routine.

Open `sprite-joy-sid-bumps.web64proj` in Web64 IDE, enable audio if the browser asks, and use joystick port 2 or the mapped gamepad/keyboard controls.
